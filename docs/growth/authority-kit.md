# Ball IQ — Authority Kit (backlink / domain-authority playbook)

**Why this exists.** GSC (2026-07-21) proved Ball IQ has a *ranking*, not indexing or demand,
problem: 349 queries pulling 2.65k impressions, but stuck at avg **position 21.8** — page 2.
55/56 pages are indexed. Content is being *seen*, not clicked. The one lever that moves *every*
page off page 2 at once is **domain authority**, and Ball IQ has almost no inbound links. This is
the untouched growth lever. Only Alex can pull it (submissions + outreach), so this is the
ready-to-execute list.

**The rule that governs all of it:** *relevance beats volume.* One link from a football site or a
Wordle-spinoff directory is worth more than fifty from a generic "500+ free directory" list —
and those spam lists can actively hurt. **Do NOT** work generic directory-dump lists.

**Link velocity:** spread submissions over **4–6 weeks**, a handful per week. A brand-new domain
that suddenly gains 100 links looks manufactured. Slow is safe and works better.

**The hook, everywhere:** lead with **Footle** (the free daily football Wordle), not "the app."
It plays in-browser in 3 seconds, needs no signup, and is the most linkable, most on-theme thing
we have. Link target = `https://balliq.app/footle`. The answer page
(`https://balliq.app/football-wordle/answer/`) is a second strong, linkable asset.

## Tier 1 — Wordle-spinoff / daily-game directories (TOPICAL GOLD — do these first)

Footle is a football Wordle. These directories exist to list exactly that, they rank well, and a
link from them is maximally *relevant*. Most add games via a submit form, an email, or a Twitter/X
DM. Pitch: *"Footle — a free daily Football (soccer) Wordle. Guess the footballer's surname in 6.
Plays in-browser, no signup: balliq.app/footle."*

| Site | How to get listed | Notes |
|---|---|---|
| **listdle.com** | Has a **Sports** category — find their submit/contact link and request inclusion | Best fit: sports daily-games directory |
| **wordly.org/wordle-games** | 398 games listed — look for "submit a game" / contact | Large, well-trafficked spinoff hub |
| **crosswordle.com/blog/wordle-variants** | Editorial "variants" list — email/DM to be added | Ranked list, gets search traffic |
| **thinkygames.com** | Curated daily-puzzle features — pitch Footle for a feature/listing | Higher editorial bar; good if accepted |
| **Medium "authoritative list of Wordle spinoffs" (floodgates)** | Comment / contact the author to be added | Author-maintained; a mention = a link |
| **parade.com "games like Wordle"** | Long shot (major outlet) — only if a PR angle appears | Lower priority |

> Search "wordle spinoffs list" / "games like wordle" periodically — new aggregators appear
> constantly and early inclusion is easy.

## Tier 2 — Product Hunt launch (one-time, high value)

A single well-run PH launch = a high-DA dofollow link + a real traffic spike + social proof.
- Launch **Footle** (the game), not "Ball IQ the app" — the free, no-signup hook converts best.
- Prep: a good tagline, 3–4 clean screenshots/GIF (the Footle grid + a share card), a first
  comment telling the origin story. Launch Tue–Thu, 00:01 PT.
- Rally the socials the morning of (Threads/IG) — but as "we're live on PH", not a hard sell.

## Tier 3 — Quality general listings (legit, worth the time)

- **AlternativeTo** — list Footle as a "Wordle alternative" and Ball IQ as a trivia-app
  alternative. High DA, genuinely used, topical.
- **Slant.co** — add Ball IQ / Footle to relevant "best football quiz / best Wordle-like" lists.
- **SaaSHub / similar product aggregators** — a clean listing, no badge shenanigans.
- **There's an "App of the Day" / indie-maker directories** (e.g. the indie lists from the DR-77+
  set): only the ones that *don't* demand a reciprocal badge on our homepage — a badge on
  balliq.app is a design/UX cost that isn't worth a single link.

## Tier 4 — Football-niche outreach (highest-value links, most effort)

Topical links from football sites are the best of all. These are relationships, not form-fills.
- **Football subreddits** — see `reference_reddit_club_subs` (memory) for the Green/Yellow/Red map
  and the modmail-first unlock. Voice = fan sharing a fun thing, never founder. The `/football-
  wordle/answer/` page and per-club quizzes are natural shares.
- **Club/football blogs & fan sites** — small ones link generously. Offer a per-club quiz link
  ("we made a [Club] quiz, thought your readers might enjoy it").
- **Football newsletters / Substacks** — a "game of the week" mention is a link + engaged traffic.
- **Football Discords** — share Footle where daily-games are welcomed (not spam).

## Outreach templates

**A. Directory / aggregator submission (email or DM)**
> Subject: A football Wordle for your list — Footle
>
> Hi — I built **Footle**, a free daily Football (soccer) Wordle: guess the footballer's surname
> in six tries. It plays in the browser, no signup, new player every day.
> Would you consider adding it to [site/list]? Link: https://balliq.app/footle
> Happy to send a screenshot or the daily share grid. Thanks either way!

**B. Football blog / newsletter (topical link)**
> Subject: A free [Club] quiz your readers might like
>
> Hi [name] — love [site]. I run Ball IQ, a football quiz site, and we built a deep **[Club]**
> quiz (deep-cut questions with explanations) plus a daily football Wordle.
> If it's a fit for your readers, here's the [Club] quiz: [/quiz/<club>/] and the daily game:
> balliq.app/footle. No ask beyond that — thought it might be fun for the audience.

**C. Reddit (comment-first, never a naked link post)**
> (Use only where the sub allows it and after building comment karma — see the reddit memory.
> Share as a fan: "made this daily football Wordle, been enjoying it — balliq.app/footle".
> No question-counts, no founder voice, no crossposting.)

- [ ] Re-check GSC avg position in ~3–4 weeks — expect the near-misses (arsenal ~15, tottenham
      ~10, fenerbahce ~14) to be the first to break onto page 1 as authority lands.

## Skip list (don't waste time / avoid harm)

- Generic "300+/500+ free directory submission sites" lists — low-quality, often spammy, Google
  discounts or penalises them.
- Any directory demanding a **badge/link back on balliq.app's homepage** — not worth a homepage
  design cost for one link.
- Paid "guest post" / link-farm offers — against Google guidelines, risk > reward.
